3. Cookies, SDKs and Similar Technologies

---
• Cookies

A cookie is a data file, often including a unique identifier, that is sent to your browser from a web server
and is then stored on the hard drive of your device.

---
• SDKs

An SDK (Software Development Kit) is a collection of software tools, libraries, documentation, code
samples, and sometimes APIs (Application Programming Interfaces) that developers use to create
applications for specific platforms, frameworks, or devices. SDKs make it easier to integrate certain
features or functionalities into an app without building them from scratch.

Our application integrates third-party Software Development Kits (SDKs) to enhance its functionality and
provide certain features. These SDKs may collect information automatically, such as:

• Device identifiers (e.g., IMEI, MAC address, advertising ID)
• Usage data (e.g., app interactions, session duration)
• Technical data (e.g., IP address, operating system, device type, mobile network information)
• Location information, if you grant permission

The data collected by these SDKs is used to improve user experience, provide analytics, serve
advertisements, or enable features like in-app purchases. We ensure that any third-party SDK we
integrate complies with applicable data protection laws and has a robust privacy policy governing the
collection and use of data.
